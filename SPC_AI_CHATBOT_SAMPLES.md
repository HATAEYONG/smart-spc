## 샘플 2: 근본 원인 분석 (Root Cause Analysis)

### 사용자 질문:
**"최근 발생한 품질 문제의 근본 원인을 알려주세요"**

### AI 응답:
```
🔍 품질 문제 근본 원인 분석

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[문제 개요]
• 발생 일시: 2025-01-12 09:23
• 제품: BOLT-M10
• 문제 유형: Rule 1 위반 (3σ 벗어남)
• 측정값: 10.85 mm (규격 초과)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[원인 분석 - 5M1E]

📌 **Man (작업자)**
• 조사 결과: 작업자 교육 이수 완료
• 문제 여부: 없음 ✓

📌 **Machine (설비)**
• 조사 결과: CNC-001 장비 온도 이상 발견
• 정상 온도: 20±2°C → 실제: 26°C
• 문제 여부: **있음** ⚠️

📌 **Material (원자재)**
• 조사 결과: 원자재 입검 합격
• 문제 여부: 없음 ✓

📌 **Method (작업 방법)**
• 조사 결과: 표준 작업 절차 준수
• 문제 여부: 없음 ✓

📌 **Measurement (측정)**
• 조사 결과: 캘리퍼 교정 유효기간 내
• 문제 여부: 없음 ✓

📌 **Environment (환경)**
• 조사 결과: 작업장 습도 정상 범위
• 문제 여부: 없음 ✓

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[근본 원인 결론]
🎯 **설비 온도 관리 부족**이 주요 원인으로 판단됩니다.

[권장 조치]
1. ✅ CNC-001 장비 냉각 시스템 점검
2. ✅ 온도 모니터링 센서 교체
3. ✅ 2시간마다 온도 체크 실시

추가 조언이 필요하시면 언제든 문의하세요! 💪
```

### 샘플 4: Western Electric Rules 설명

### 사용자 질문:
**"Western Electric Rules가 뭔가요?"**

### AI 응답:
```
📚 Western Electric Rules 설명

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Western Electric Rules는 관리도에서 공정 이상을
조기에 탐지하기 위한 8가지 검출 규칙입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[8가지 규칙]

🔴 **Rule 1: 관리한계 벗어남**
   • 1개 점이 UCL 또는 LCL을 벗어남
   • 심각도: ⭐⭐⭐⭐ (매우 높음)

🟠 **Rule 2: 9개 연속 동일측**
   • 9개 연속 점이 중심선 같은 쪽에 위치
   • 심각도: ⭐⭐⭐ (높음)

🟡 **Rule 3: 6개 연속 증가/감소**
   • 6개 연속 점이 지속적으로 증가하거나 감소
   • 심각도: ⭐⭐⭐ (높음)

🟢 **Rule 4: 14개 교차 패턴**
   • 14개 연속 점이 상하로 교차
   • 심각도: ⭐⭐ (중간)

🔵 **Rule 5: 4개 중 3개가 2σ 벗어남**
   • 연속 3개 중 2개가 2σ 영역 밖
   • 심각도: ⭐⭐ (중간)

🟣 **Rule 6: 6개 중 5개가 1σ 벗어남**
   • 연속 5개 중 4개가 1σ 영역 밖
   • 심각도: ⭐ (낮음)

⚪ **Rule 7: 15개 연속 1σ 이내**
   • 15개 연속 점이 ±1σ 이내에 위치
   • 심각도: ⭐ (낮음)
   • 의미: 분산 감소 또는 데이터 조작 가능

⚫ **Rule 8: 8개 연속 1σ 밖**
   • 8개 연속 점이 ±1σ 밖에 위치
   • 심각도: ⭐⭐ (중간)
   • 의미: 혼합 또는 층화

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[활용 방법]
✅ 공정 안정성 모니터링
✅ 이상 원인 조기 경보
✅ 품질 개선 기회 식별
✅ 공정 능력 향상

더 자세한 내용이 궁금하시면 언제든 질문해주세요! 😊
```

## 응답 포맷 가이드라인

### ✅ 모든 응답에 포함할 요소

1. **헤더 섹션**
   - 이모지 아이콘 (📊, 🔍, 🚀 등)
   - 명확한 제목

2. **구분선**
   - `━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━` 사용

3. **데이터/정보 섹션**
   - 불렛 포인트 또는 숫자 매기기
   - 명확한 섹션 구분 ([제목])

4. **시각적 강조**
   - 중요한 내용: **굵은 글씨**
   - 성공: ✅
   - 경고: ⚠️
   - 실패/위험: 🔴

5. **마무리**
   - 친근한 메시지
   - 추가 질문 유도
   - 이모지 (😊, 💪, 🌟 등)

### 🎨 색상 및 스타일

- 보초색: 6자리 정수/값은 명확하게 표시
- 위험도: ⭐ (1-4개) 또는 이모지로 표현
- 상태: 색상 원 또는 체크박스로 시각화
- 중요 수치: **굵은 글씨**로 강조
